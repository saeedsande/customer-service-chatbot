# 🤖 SmartAssist – AI Customer Support Platform

A modern AI-powered customer support chatbot built using Python, Flask, HTML, CSS, and JavaScript. SmartAssist helps users quickly resolve common customer service queries such as order tracking, refunds, shipping information, account management, payment support, and more.

---

## 🚀 Overview

SmartAssist is a customer service automation platform designed to improve user support experiences through conversational AI and NLP-based intent matching.

The application provides an interactive support dashboard with a modern SaaS-inspired interface, enabling users to get instant responses to common support requests.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/saeedsande/customer-service-chatbot.git
```

### Navigate to Project

```bash
cd customer-service-chatbot
```

### Install Dependencies

```bash
pip install flask
```

### Run Application

```bash
python app.py
```

### Open Browser

```bash
http://127.0.0.1:5000
```
